Arbetsflöde/gång

  1. skapa en fork av ett respository
  2. klonade repot med github desktop (gröna knappen)
  3. Öppnade report visual studio code
  4. öppnade filen index.html
  5. Ändrade i index.html, skapade listor och bild
  6. för att visa bilden så gjorde vi images mapp och sparade bilden i den
  7. Sedan använde vi img taggen med src attributet för att få bilden att laddas
  8. Bilden va gigantiskt, vi öppnade den i paint och minskade bilden
